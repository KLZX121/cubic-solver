Cubic Equation Solver

Solve any function of the form y = ax^3 + bx^2 + cx + d given 4 points, using simultaneous equations which are solved with Gaussian Elimination. Due to floating point error, digits in small decimal places can be inaccurate, as seen when inputting the example data for y=x^2+3. To mitigate this, enable the rounding option if high accuracy is not required.

Example data inputs:

y=x: (1,1),(-2,-2),(3,3),(-7,-7)
y=x^2+3: (-1,4),(2,7),(-3,12),(4,19)
y=x^3-x: (0,0),(1,0),(-1,-2),(2,4)
y=x^3+x^2+x+1: (0,1),(-1,0),(1,4),(-2,-5)

Input your points as (x1,y1),(x2,y2),(x3,y3),(x4,y4):

Round final equation to 3 significant figures
